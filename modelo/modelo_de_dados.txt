-------------------------------------------------
                     MÉDICOS
-------------------------------------------------
- _id (ObjectId)
- nome (String)
- data_nascimento (Date)
- especialidades (Array de Strings) → Ex.: ["Pediatria", "Clínica Geral"]
- tipo (String) → "Generalista", "Especialista" ou "Residente"
- contato:
    - telefone (String)
    - email (String)
- documentos:
    - CPF (String)
    - RG (String)
- em_atividade (Boolean)

-------------------------------------------------
                    PACIENTES
-------------------------------------------------
- _id (ObjectId)
- nome (String)
- data_nascimento (Date)
- endereco (String)
- telefone (String)
- email (String)
- documentos:
    - CPF (String)
    - RG (String)
- convenio_id (ObjectId) → referência para coleção Convenios

-------------------------------------------------
                    CONVÊNIOS
-------------------------------------------------
- _id (ObjectId)
- nome (String)
- cnpj (String)
- carencia (String) → exemplo: "30 dias"

-------------------------------------------------
                    CONSULTAS
-------------------------------------------------
- _id (ObjectId)
- data_hora (Date)
- medico_id (ObjectId)
- paciente_id (ObjectId)
- valor (Double)
- convenio_id (ObjectId) (opcional)
- num_carteira (String) (opcional)
- especialidade (String)
- receituario:
    - medicamento (String)
    - quantidade (String)
    - instrucoes (String)
  (pode ser um array com mais de 1 medicamento)

-------------------------------------------------
                   INTERNAÇÕES
-------------------------------------------------
- _id (ObjectId)
- data_entrada (Date)
- data_prev_alta (Date)
- data_efetiva_alta (Date)
- procedimentos (String)
- quarto_id (ObjectId)
- medico_id (ObjectId)
- paciente_id (ObjectId)
- enfermeiros_ids (Array de ObjectId)

-------------------------------------------------
                     QUARTOS
-------------------------------------------------
- _id (ObjectId)
- numero (Int)
- tipo (String) → "Apartamento", "Duplo", "Enfermaria"
- valor_diaria (Double)

-------------------------------------------------
                   ENFERMEIROS
-------------------------------------------------
- _id (ObjectId)
- nome (String)
- cpf (String)
- coren (String)
